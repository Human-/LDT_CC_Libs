Field(s)

textutils.formatTime(time, TwentyFourHour)

Converts os#time to a human readable string.

Example results:

  1. 5:30 PM if TwentyFourHour is false.
  2. 17:30 if TwentyFourHour is true.

Parameters

  • os#time time : time to be formatted.

  • #boolean TwentyFourHour : if true, time will be formatted in 24 hour format instead of 12 hour.

Return value

#string: the human readable string.

textutils.pagedPrint(text, confirmation_lines)

Prints a string onto the screen, but waits for confirmation before scrolling down.

Parameters

  • #string text : string to be printed.

  • #string confirmation_lines : string prompt to be displayed when awaiting confirmation.

Return value

#string: string printed on the screen.

textutils.serialize(t)

Returns a string representation of the table t that can be saved or sent over rednet.

Notes:

  1. Nested structures are supported, but tables with circular graphs will be detected and raise an error.

Parameter

  • #table t : table to be serialized into a string.

Return value

#string: serialized string form of table t.

textutils.slowPrint(text, rate)

Slowly prints a string to the screen.

Notes:

  1. Unlike slowWrite, it prints a newline at the end of the string.

Parameters

  • #string text : string to be displayed.

  • #number rate : rate at which the string will be displayed.

Return value

#string: string printed on the screen.

textutils.slowWrite(text, rate)

Slowly writes a string to the screen.

Notes:

  1. Unlike slowPrint, it does not add a newline to the end of the string.

Parameters

  • #string text : string to be written.

  • #number rate : rate at which the string will be written.

Return value

#string: string written on the screen.

textutils.tabulate(...)

Prints tables in an ordered form.

Each table's row and column width is auto-adjusted.

Parameter

  • ... : list of tables to be printed.

Return value

#string: string printed on the screen.

textutils.unserialize(s)

Returns a table reassembled from the string s.

Notes:

  1. The representation used by textutils actually creates Lua-friendly syntax, so that loadstring() will also work to unserialize.

Parameter

  • #string s : serialized string form of a table.

Return value

#table: table reconstructed from string s.

textutils.urlEncode(url)

Replaces certain characters in a string to make it safe for use on the internet.

Notes:

  1. This is a string function and will even replace characters that are allowed in URLs.
  2. Use this only for string's to be inserted in an URL, not the URL itself.

Parameter

  • #string url : url string to be safely encoded for transmission.

Return value

#string: A string safe for use on the internet.
