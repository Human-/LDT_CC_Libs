Type shell

Field(s)

shell.aliases()

Returns a table of all set aliases.

Return value

#table: table of all aliases currently in use.

shell.clearAlias(alias)

Removes an alias to a program.

Parameter

  • #string alias : previously set alias to remove.

shell.dir()

Returns the path to the active directory.

Return value

fs#path: path of the active directory.

shell.exit()

Exits the current program (or shell if no program is open).

Return value

#string: string displayed on the screen.

shell.getRunningProgram()

Returns the path of the currently running shell or program.

Return value

fs#path: path of the currently running shell or program.

shell.path()

Returns a string containing multiple paths separated by colons.

It is used to locate programs when no absolute path is stated.

Examples:

  1. Running id looks for the id program under each path returned by this function (by default, ./id then /rom/programs/id).
  2. Running /rom/programs/id on the other hand only checks the location specified.

Return value

#string: : separated list of paths.

shell.programs(IncludeHidden)

Returns a sorted table of all programs found along shell.path.

Parameter

  • #boolean IncludeHidden : if false, hidden programs remain hidden.

Return value

#table: table of all programs available to run.

shell.resolve(localpath)

Resolves a string path relative to shell.dir to an absolute path.

Notes:

  1. Does nothing if the path is already absolute.
  2. Uses fs#fs.combine internally.
  3. Does not verify if the path actually exists (or is even valid).

Parameter

  • #string localpath : string or path to convert to a path.

Return value

fs#path: an absolute path to a given file system resource.

shell.resolveProgram(name)

Resolves the absolute path to the program whose name you provided.

Notes:

  1. Accounts for any aliases set.
  2. Looks along the shell.path as needed.

Parameter

  • #string name : filename or alias of the requested program.

Return values

  1. fs#path: path to the requested program.

  2. #nil: nil if the program path could not be located.

shell.run(either, ...)

Runs the specified program with the specified arguments

Parameters

  • #string either : the name or path of the program to be run

  • ... : arguments for the program to be run.

Return value

#boolean: false on error, otherwise true.

shell.setAlias(alias, command)

Sets an alias for a given program.

Notes:

  1. Allows access to a program via multiple names.
  2. Default aliases include: mv for move, dir for list, ls for list, cp for copy, and rm for delete.

Parameters

  • #string alias : string to add as an alias.

  • #string command : program that is being aliased.

shell.setDir(path)

Sets the active directory to the given path.

Parameter

  • fs#path path : path to use as the active directory.

shell.setPath(path)

Sets the paths used to locate programs when no absolute path is stated.

Notes:

  1. May contain multiple paths separated by a :.
  2. Default value: ".:/rom/programs"

Parameter

  • #string path : : separated list of paths.
