Module term

Terminal API

The term API provides functions for writing text to the terminal and monitors, and drawing ASCII graphics.

Notes:

  1. Color/Colour setting functions are only available on advanced computers and monitors.

Type term

term.clear()

Clears the entire screen.
term.clearLine()

Clears the line the cursor is on.
term.getCursorPos()

Returns two arguments containing the x and the y position of the cursor.
term.getSize()

Returns the width and height of the terminal window.
term.isColor()

Returns if the computer supports color.
term.isColour()

Returns if the computer supports colour.
term.native

Internal.

Redirects terminal output to a monitor or other redirect target.
term.restore()

Restores terminal output to the previous target.

Scrolls the terminal n lines.

Type term

Field(s)

term.clear()

Clears the entire screen.

term.clearLine()

Clears the line the cursor is on.

term.getCursorPos()

Returns two arguments containing the x and the y position of the cursor.

Return value

#number, #number: The x,y coordinates of the cursor.

term.getSize()

Returns the width and height of the terminal window.

Notes:

  1. Good for if you're making something to be compatible with both Turtles and Computers.

Return value

#number, #number: width and height of the terminal.

term.isColor()

Returns if the computer supports color.

Notes:

  1. Used to determine whether or not an Advanced Computer is being used.

Return value

#boolean: true if the Computer is Advanced.

term.isColour()

Returns if the computer supports colour.

Notes:

  1. Used to determine whether or not an Advanced Computer is being used.

Return value

#boolean: true if the Computer is Advanced.

#table term.native

Internal.

Notes:

  1. Copy of the original terminal environment for use with term.restore()
term.redirect(target)

Redirects terminal output to a monitor or other redirect target.

Notes:

  1. Use peripheral.wrap to acquire a monitor "object".

Parameter

  • #table target : The wrapped peripheral (ie. monitor).

term.restore()

Restores terminal output to the previous target.

term.scroll(n)

Scrolls the terminal n lines.

Parameter

  • #number n : Lines to scroll.

term.setCursorBlink(state)

Sets whether the cursor is blinking or not.

Parameter

  • #boolean state : true to enable cursor blinking, false to disable.

term.setCursorPos(x, y)

Sets the location of the cursor on screen.

Parameters

  • #number x : the new x coordinate of the cursor.

  • #number y : the new y coordinate of the cursor.

term.write(text)

Writes a string to the screen.

Parameter

  • #string text : The string to be written.

Return value

#string: string written on the screen.
