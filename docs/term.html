Module term

The Terminal API provides functions for writing text to the terminal and monitors, and drawing ASCII graphics.
Color/Colour setting functions are only available on advanced computers and monitors.

Type term

term.clear() Clears the entire screen
term.clearLine() Clears the line the cursor is on
term.getCursorBlink(bool) Disables the blinking or turns it on.
term.getCursorPos() Returns two arguments containing the x and the y position of the cursor.
term.getSize() Returns two arguments containing the x and the y values stating the size of the screen.
term.isColor() Returns if the computer supports color.
term.isColour() Returns if the computer supports colour.
term.redirect(target) Redirects terminal output to a monitor or other redirect target.
term.restore() Restores terminal output to the previous target.
term.scroll(n) Scrolls the terminal n lines.
term.setBackgroundColor(color) Sets the background-color of the terminal.
term.setBackgroundColour(colour) Sets the background-colour of the terminal.
term.setTextColor(color) Sets the text-color of the terminal.
term.setTextColour(colour) Sets the text-colour of the terminal.
term.write(text) Writes text to the screen.

Type term

Field(s)

term.clear()
Clears the entire screen
term.clearLine()
Clears the line the cursor is on
term.getCursorBlink(bool)
Disables the blinking or turns it on.

Parameter

  • #boolean bool : True if the cursor shall be enabled
term.getCursorPos()
Returns two arguments containing the x and the y position of the cursor.

Return values

  1. #number: The x coordinate of the Cursor
  2. #number: The y coordinate of the Cursor
term.getSize()
Returns two arguments containing the x and the y values stating the size of the screen.
(Good for if you're making something to be compatible with both Turtles and Computers).

Return values

  1. #number: Width of the terminal
  2. #number: Height of the terminal
term.isColor()
Returns if the computer supports color.
(Used to determine whether or not an Advanced Computer is being used).

Return value

#boolean: True if the Computer is Advanced

term.isColour()
Returns if the computer supports colour.
(Used to determine whether or not an Advanced Computer is being used).

Return value

#boolean: True if the Computer is Advanced

term.redirect(target)
Redirects terminal output to a monitor or other redirect target.
(Use peripheral.wrap to acquire a monitor "object").

Parameter

  • global#device target : The wrapped peripheral (ie. monitor)
term.restore()
Restores terminal output to the previous target.
term.scroll(n)
Scrolls the terminal n lines.

Parameter

  • #number n : Lines to scoll
term.setBackgroundColor(color)
Sets the background-color of the terminal.
Available only to Advanced Computers and Advanced Monitors.

Parameter

  • #number color : the color to use
term.setBackgroundColour(colour)
Sets the background-colour of the terminal.
Available only to Advanced Computers and Advanced Monitors.

Parameter

  • #number colour : the colour to use
term.setTextColor(color)
Sets the text-color of the terminal.
Available only to Advanced Computers and Advanced Monitors.

Parameter

  • #number color : The color to use
term.setTextColour(colour)
Sets the text-colour of the terminal.
Available only to Advanced Computers and Advanced Monitors.

Parameter

  • #number colour : The colour to use
term.write(text)
Writes text to the screen.

Parameter

  • #string text : The text to write

Return value

#string: Text written on the screen
