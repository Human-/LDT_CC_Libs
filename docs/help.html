Module help

Help API

The help API allows limited access to the help files.

Type help

help.lookup(topic)

Looks up the path for a help file using the help.path directory.
help.path()

Returns the current path for the help API.
help.setPath(apiPath)

Sets the path of the API to apiPath.
help.topics()

Returns a table of valid help topics.

Type help

Field(s)

help.lookup(topic)

Looks up the path for a help file using the help.path directory.

Parameter

  • #string topic : string topic to look up for help.

Return values

  1. fs#path: path of help file for topic.

  2. #nil: nil if help file for topic is not found.

help.path()

Returns the current path for the help API.

Return value

fs#path: path of help API.

help.setPath(apiPath)

Sets the path of the API to apiPath.

Parameter

  • fs#path apiPath : path to set the API to.

help.topics()

Returns a table of valid help topics.

Return value

#table: all valid topics.
