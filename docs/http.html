Module http

HTTP API

The http API allows interfacing with websites and downloading from them.

Notes:

  1. The HTTP API must be enabled in ComputerCraft.cfg before being used. To enable it open /config/ComputerCraft.cfg and change enableAPIhttp=false to enableAPIhttp=true.

Type http

http.get(url)

Sends a HTTP GET request to a website.
http.post(url, postData)

Sends a HTTP POST request to a website.
http.request(url, ...)

Sends a HTTP request to the website.

Type http

Field(s)

http.get(url)

Sends a HTTP GET request to a website.

Returns a table similar to that returned by an fs#fs.open() call, used to handle the response of the server on success or nil on failure.

Parameter

  • #string url : the url to query.

Return values

  1. #table: the page source of the requested resource.

  2. #nil: nil on failure.

Usage:

local sExample = http.get("http://example.com/") --Get contents of page
write(sExample.readAll()) --Read and print contents of page
sExample.close() --Just in case
http.post(url, postData)

Sends a HTTP POST request to a website.

Return the response of the server or nil on failure.

Parameters

  • #string url : the url to query.

  • #string postData : any POST data required by the website.

Return values

  1. #table: the server response.

  2. #nil: nil on failure.

http.request(url, ...)

Sends a HTTP request to the website.

Notes:

  1. Returns event "http_success" on success.
  2. Returns event "http_failure" on failure.

Parameters

  • #string url : the url to query.

  • ... : postData any data required by the website.

Usage:

http.request("http://pastebin.com/raw.php?i=Tk19jv43")
local requesting = true
while requesting do
   local event, url, sourceText = os.pullEvent()
   if event == "http_success" then
      local respondedText = sourceText.readAll()
      print(respondedText)
      requesting = false
   elseif event == "http_failure" then
      print("Server didn't respond.")
      requesting = false
   end
end
