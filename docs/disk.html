Module disk

Disk API

The disk API allows you to interact with Disk Drives.

Type disk

disk.eject(side)

Ejects the Disk on side.
disk.getAudioTitle(side)

Returns the audio title used by the Disk on side.
disk.getID(side)

Gets diskID used by the Disk on side.
disk.getLabel(side)

Gets the string label used by the Disk on side.
disk.getMountPath(side)

Returns the directory path on which the Disk on side has been mounted.
disk.hasAudio(side)

Checks whether the Disk on side is a Music Disk, as opposed to a Floppy Disk.
disk.hasData(side)

Checks whether the Disk on a given side is a Floppy Disk, as opposed to a Music Disk.
disk.isPresent(side)

Checks the given side for a Disk.
disk.playAudio(side)

Plays the audio contained in the Disk on side.
disk.setLabel(side, label)

Sets the string label used by the Disk on side.
disk.stopAudio(side)

Stops the audio playing from the Disk on side (if any).

Type disk

Field(s)

disk.eject(side)

Ejects the Disk on side.

Parameter

  • peripheral#side side : the side of the computer to consider.

disk.getAudioTitle(side)

Returns the audio title used by the Disk on side.

Parameter

  • peripheral#side side : the side of the computer to consider.

Return value

#string: the title used by the Disk.

disk.getID(side)

Gets diskID used by the Disk on side.

Parameter

  • peripheral#side side : the side of the computer to consider.

Return value

#diskID: the diskID of the Disk.

disk.getLabel(side)

Gets the string label used by the Disk on side.

Parameter

  • peripheral#side side : the side of the computer to consider.

Return value

#string: the string label used by the Disk.

disk.getMountPath(side)

Returns the directory path on which the Disk on side has been mounted.

Example:

  1. /disk

Parameter

  • peripheral#side side : the side of the computer to consider.

Return value

fs#path: path of the mounted Disk.

disk.hasAudio(side)

Checks whether the Disk on side is a Music Disk, as opposed to a Floppy Disk.

Parameter

  • peripheral#side side : the side of the computer to consider.

Return value

#boolean: true if the Disk has saved audio, otherwise false.

disk.hasData(side)

Checks whether the Disk on a given side is a Floppy Disk, as opposed to a Music Disk.

Parameter

  • peripheral#side side : the side of the computer to consider.

Return value

#boolean: true if the disk has saved data, otherwise false.

disk.isPresent(side)

Checks the given side for a Disk.

Parameter

  • peripheral#side side : the side of the computer to consider.

Return value

#boolean: true if there is a Disk, otherwise false.

disk.playAudio(side)

Plays the audio contained in the Disk on side.

Parameter

  • peripheral#side side : the side of the computer to consider.

disk.setLabel(side, label)

Sets the string label used by the Disk on side.

Parameters

  • peripheral#side side : the side of the computer to consider.

  • #string label : the string label to be used.

disk.stopAudio(side)

Stops the audio playing from the Disk on side (if any).

Parameter

  • peripheral#side side : the side of the computer to consider.

Type diskID

The unique ID of a Disk.

Implemented as a #number.
