Module paintutils

Paintutils API

The paintutils API allows for loading and drawing images.

Notes:

  1. paintutils is only available for Advanced Computers and Advanced Monitors.

Type paintutils

paintutils.drawImage(image, x, y)

Draws an image at (x, y) where image is an image object.
paintutils.drawLine(startX, startY, endX, endY, color)

Draws a line from (startX, startY) to (endX, endY) in the color/colour specified.
paintutils.drawPixel(x, y, color)

Draws a pixel at (x, y) in the color/colour specified.
paintutils.loadImage(path)

Returns an image object.

Type paintutils

Field(s)

paintutils.drawImage(image, x, y)

Draws an image at (x, y) where image is an image object.

Parameters

  • #image image : image to be drawn.

  • #number x : x coord to draw image at.

  • #number y : y coord to draw image at.

Return value

#boolean: true on success, otherwise false.

paintutils.drawLine(startX, startY, endX, endY, color)

Draws a line from (startX, startY) to (endX, endY) in the color/colour specified.

Parameters

  • #number startX : x coord to begin drawing the line.

  • #number startY : y coord to begin drawing the line.

  • #number endX : x coord to stop drawing the line.

  • #number endY : y coord to stop drawing the line.

  • colors#color color : colors/colours of the line.

Return value

#boolean: true on success, otherwise false.

paintutils.drawPixel(x, y, color)

Draws a pixel at (x, y) in the color/colour specified.

Parameters

  • #number x : x coord to draw pixel at.

  • #number y : y coord to draw pixel at.

  • colors#color color : colors/colours of the pixel.

Return value

#boolean: true on success, otherwise false.

paintutils.loadImage(path)

Returns an image object.

Parameter

  • fs#path path : file path to the image.

Return value

#image: image object.
