Global API

GPS API

The gps API provides a method for turtles and computers to retrieve their own locations.

Help API

HTTP API

The http API allows interfacing with websites and downloading from them.
io

IO API

The io API is a reimplementation of the default Lua input / output API with some features trimmed.
keys

Keys API

The keys API contains constants for all the key codes that can be returned by the "key" event:

Example usage:

local sEvent, nKey = os.pullEvent()
if sEvent == "key" and nKey == keys.enter then
   Do something
end
math

Mathmatics API

The math API is an interface to the standard C math library.

The os API allows for interfacing with the Lua based Operating System itself.

Paintutils API
