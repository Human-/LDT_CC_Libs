os.loadAPI(path)

Loads a Lua script as an API in its own namespace.

Notes:

  1. It will be available to all programs that run on the terminal.

Parameter

  • fs#path path : path to the API file to load.

os.pullEvent(target)

Blocks until the Computer receives an event, or if target-event is specified, will block until an instance of target-event occurs.

Notes:

  1. If a target-event is specified, the computer will not break for any other events (except termination).

Events:

  • char (string letter)
  • key (keys#keycode keycode)
  • timer (#timerID timerID)
  • alarm (#alarmID alarmID)
  • redstone
  • terminate
  • disk (string side)
  • disk_eject (string side)
  • peripheral (string side)
  • peripheral_detach (string side)
  • rednet_message (#computerID senderID, string message, number distance)
  • http_success (string url, table text)
  • http_failure (string url)
  • mouse_click (keys#keycode mouse button, number x coord, number y coord)
  • mouse_scroll (number direction, number x coord, number y coord)
  • mouse_drag (keys#keycode mouse button , number x coord, number y coord)
  • monitor_touch (string side, number x coord, number y coord)

Parameter

  • #string target : event Only accept the specified event.

Return value

#string, #Untyped: Returns event type and any event parameters.

os.pullEventRaw()

Advanced version of pullEvent().

Notes:

  1. Unlike os.pullEvent( target-event ), this function will not break for system events, and can be used to handle termination events.

Return value

#string, #Untyped: Returns event type and any event parameters.

os.queueEvent(eventName, ...)

Adds an event eventName with the specified parameters to the event queue.

Events:

  • redstone

  • mouse_drag (keys#keycode mouse button , number x coord, number y coord)
  • monitor_touch (string side, number x coord, number y coord)

Parameters

  • #string eventName :

  • ... : any event parameters

os.reboot()

Reboots the computer.

os.run(envVars, path, ...)

An advanced way of starting programs.

A started program will have a given environment table which determines what functions it has available, as well as any variables it will be able to access by default.

Notes:

  1. You may prefer to use the shell (API) unless you need to do something special.

Parameters

  • #table envVars : environment variables for the program to run under.

  • fs#path path : path to the program to be run.

  • ... : arguments for the program to be run.

Return value

#boolean: false on error, otherwise true.

os.setAlarm(time)

Adds an alarm which will fire an alarm event at the specified Minecraft gametime.

Notes:

  1. The alarmID is returned from this function to differentiate multiple alarms.

Parameter

  • #gametime time : Minecraft world time to fire the event.

Return value

#alarmID: alarmID

os.setComputerLabel(label)

Set the string label of this Computer.

Parameter

  • #string label : the string label to be used by the Computer.

os.shutdown()

Turns off the computer.

os.sleep(time)

Makes the system wait a number of seconds before continuing in the program.

Notes:

  1. os.sleep( timeout ) may also be used as simply sleep( timeout ).
  2. Fractions of a second are supported, but only down to a game tick, or 1/20 of a second (0.05s). times are rounded up to the next tick.

Parameter

  • #time time : Seconds to pass before resuming.

os.startTimer(time)

Adds a timer which will fire a timer event once after time seconds have passed.

Notes:

  1. The timerID of the timer is returned from this function to differentiate multiple timers.

Parameter

  • #time time : Seconds to pass before the event fires.

Return value

#timerID: timerID

os.time()

Returns the current Minecraft world gametime.

Return value

#gametime: The current gametime.

os.unloadAPI(path)

Unloads a previously loaded API.

Parameter

  • fs#path path : path to the API file to load.

os.version()

Returns the string version of the OS that the Computer is running.

Notes:

  1. Returns "CraftOS " .. osVersion on a Computer.
  2. Returns "TurtleOS " .. osVersion on a Turtle.
  3. May return other strings if a modded OS is loaded.

Return value

#string: the version string.
