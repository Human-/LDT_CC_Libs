Module colours

RedPower Bundled Cable Colour API The Colours API allows you to utilize Bundled Cables from the RedPower mod, and colours on Advanced Computers and Advanced Monitors.
For the non-American English version just replace 'colour' with 'color' and it will use the other API, colors - which is exactly the same, except in American English (e.g. grey is spelled gray and lightGrey is spelled lightGray). See http://computercraft.info/wiki/Colors_%28API%29 for more info

Type colours

colours.combine(...) Combines a set of colours (or sets of colours) into a larger set.
colours.subtract(colors, ...) Removes one or more colours (or sets of colours) from an initial set.
colours.test(colors, color) Tests whether a given colour (or set of colours) is contained within a set (first arguement).

Type colours

Field(s)

colours.combine(...)
Combines a set of colours (or sets of colours) into a larger set.

Parameter

  • ... : the colour(s) to use. An element of the table colours can also be provided.

Return value

#number: the set of combined colours

colours.subtract(colors, ...)
Removes one or more colours (or sets of colours) from an initial set.

Parameters

  • #number colors : the set of colours to remove from.
  • ... : colour(s) to remove from the set. An element of the table "colours" can also be provided.

Return value

#number: the set of colours with specified ones removed.

colours.test(colors, color)
Tests whether a given colour (or set of colours) is contained within a set (first arguement).

Parameters

  • #number colors : the set of colours to test with
  • #number color : the colour to check for

Return value

#boolean: whether the colour is contained in the set
