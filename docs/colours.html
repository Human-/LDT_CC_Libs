Module colours

RedPower Bundled Cable Colour API

The colours API allows you to utilize Bundled Cables from the RedPower mod, and colors on Advanced Computers and Advanced Monitors.

Notes:

  1. For the American English version just replace 'colour' with 'color' and it will use the other API, colors - which is exactly the same, except in American English (e.g. grey is spelled gray and lightGrey is spelled lightGray).
  2. Since the #colours are just integers, the standard addition (+) and subtraction (-) operators may be used on #colours instead of colours.combine and colours.subtract.
  3. Furthermore, the Bit (API) may be used on sets of #colours.

colours.subtract(colors, ...)

Removes one or more colours (or sets of colours) from an initial set.

Parameters

  • #colour colors : the set of colours to remove from.

  • ... : colour(s) to remove from the set. An element of the table "colours" can also be provided.

Return value

#colour: the set of colours with specified ones removed.

colours.test(colors, color)

Tests whether a given colour (or set of colours) is contained within a set (first arguement).

Parameters

  • #colour colors : the set of colours to test with

  • #colour color : the colour to check for

Return value

#boolean: whether the colour is contained in the set
