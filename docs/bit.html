Module bit

Bit API

The bit API is for manipulating numbers using bitwise binary operations.

Type bit

bit.band(m, n)

Computes the bitwise AND of two numbers
bit.blogic_rshift(n, bits)

Shifts a number right by a specified number of bits (zero fill >>>)
bit.blshift(n, bits)

Shifts a number left by a specified number of bits
bit.bnot(n)

Computes the bitwise NOT of a number, taken in the domain and range of 32-bit unsigned integers
bit.bor(m, n)

Computes the bitwise inclusive OR of two numbers
bit.brshift(n, bits)

Shifts a number right by a specified number of bits
bit.bxor(m, n)

Computes the bitwise exclusive OR of two numbers

Type bit

Field(s)

bit.band(m, n)

Computes the bitwise AND of two numbers

Parameters

  • #number m : the number to AND'ed

  • #number n : the number to AND'ed

Return value

#number: the value of m AND n

bit.blogic_rshift(n, bits)

Shifts a number right by a specified number of bits (zero fill >>>)

Parameters

  • #number n : the number to be manipulated

  • #number bits : the number of bits to shift n

Return value

#number: the value of n shifted right by bits bits, which is equivalent to n÷2^bits

bit.blshift(n, bits)

Shifts a number left by a specified number of bits

Parameters

  • #number n : the number to be manipulated

  • #number bits : the number of bits to shift n

Return value

#number: the value of n shifted left by bits bits, which is equivalent to n×2^bits

bit.bnot(n)

Computes the bitwise NOT of a number, taken in the domain and range of 32-bit unsigned integers

Parameter

  • #number n : the number to AND'ed

Return value

#number: the value of NOT n

bit.bor(m, n)

Computes the bitwise inclusive OR of two numbers

Parameters

  • #number m : the number to OR'ed

  • #number n : the number to OR'ed

Return value

#number: the value of m OR n

bit.brshift(n, bits)

Shifts a number right by a specified number of bits

Parameters

  • #number n : the number to be manipulated

  • #number bits : the number of bits to shift n

Return value

#number: the value of n shifted right by bits bits, which is equivalent to n÷2^bits

bit.bxor(m, n)

Computes the bitwise exclusive OR of two numbers

Parameters

  • #number m : the number to XOR'ed

  • #number n : the number to XOR'ed

Return value

#number: the value of m XOR n
