Module parallel

Parallel API

The parallel API is an API that allows you to multitask.

Type parallel

parallel.waitForAll(...)

Runs all the functions at the same time, and stops when all of them have returned.
parallel.waitForAny(...)

Runs all the functions at the same time, and stops when any of them returns.

Type parallel

Field(s)

parallel.waitForAll(...)

Runs all the functions at the same time, and stops when all of them have returned.

Parameter

  • ... : list of functions to run in parallel.

Return value

#number: A number indicating which function completed based on argument order.

parallel.waitForAny(...)

Runs all the functions at the same time, and stops when any of them returns.

Parameter

  • ... : list of functions to run in parallel.

Return value

#number: A number indicating which function completed based on argument order.
